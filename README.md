## chromeの設定(windows)
テクスチャ画像を読み込むために，以下の起動オプションを付ける．

1．chrome.exeのショートカットを作成

2. リンク先の末尾にオプションを追加

```bash
"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --allow-file-access-from-files
```

これでテクスチャを読み込めるようになります．
